# focus-garden-app
Project Name: Focus Garden
Description:
Focus Garden is a web-based productivity application designed to help users stay focused on studying or working. It uses a gamified “virtual garden” concept, where users grow plants by maintaining focus. Each completed focus session contributes to the growth of the garden, creating a positive feedback loop that encourages consistency and reduces distractions.
Objectives
Improve user focus and time management
Reduce procrastination through gamification
Provide a lightweight and accessible productivity tool
